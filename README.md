# 🎓 CampusIQ — Academic Performance & Student Intelligence Dashboard

> A real-time, browser-based academic analytics dashboard for **Sri Shanmugha College of Engineering and Technology (SSCET)** that consolidates 1,385 student records across 7 departments into a unified interactive interface with KPI cards, 6 chart visualizations, dynamic filters, and individual student profile search.

---

## 📌 Project Overview

This mini-project was developed by the **Department of AI & DS** at SSCET to solve the problem of scattered student data stored across multiple Excel files. The dashboard merges all departmental data into a single JSON dataset and displays it through an interactive web interface — enabling faculty and administrators to monitor student performance, attendance, placement eligibility, and internship status at a glance.


---

## 🚀 Features

- 📊 **8 KPI Cards** — Total students, placement eligibility %, average CGPA, average attendance, arrear count, paid internships, total certifications, below-75% attendance alert
- 🏛️ **7 Department Cards** — Per-department summary of students, CGPA, attendance, placement %, and arrears
- 📈 **6 Interactive Charts** (Chart.js)
  - Students per Department (Bar)
  - CGPA Distribution (Bar)
  - Attendance Bands (Doughnut)
  - Placement Eligibility (Doughnut)
  - Internship Status (Pie)
  - Gender Split (Doughnut)
- 🔍 **Search** — Search any student by name or SIN number and view their full profile card
- 🎛️ **Dynamic Filters (Slicers)** — Filter by Department, Year, Section, Gender, Placement status, Internship status
- 👤 **Student Profile Panel** — Detailed view with semester-wise CGPA, attendance, internal marks, and personal info
- ⚡ **Live Clock** — Real-time clock displayed in the header
- 📱 **Responsive Design** — Works on desktop and mobile browsers

## ⚙️ How to Run Locally

### Step 1 — Clone the repository
```bash
git clone https://github.com/your-username/campusiq-dashboard.git
cd campusiq-dashboard
```

### Step 2 — Install Python dependencies
```bash
pip install pandas openpyxl
```

### Step 3 — Generate the JSON dataset
Place all 7 Excel files inside the `data/` folder, then run:
```bash
python merge_data.py
```
This creates `students_data.json` with all 1,385 student records.

### Step 4 — Run with Live Server
Open the project folder in **VS Code**, install the **Live Server** extension (by Ritwick Dey), then right-click `index.html` → **Open with Live Server**.

> ⚠️ **Do not** open `index.html` by double-clicking it. Browsers block local `fetch()` requests when opened as a `file://` URL. Always use Live Server or any local web server.

## 🗂️ Data Fields

Each student record contains the following fields:

| Field | Description |
|-------|-------------|
| `SIN_No` | Student Identification Number |
| `Student_Name` | Full name |
| `Department` | Department name |
| `Year` | Academic year (II / III / IV) |
| `Section` | Section (A / B / C) |
| `Gender` | Male / Female |
| `Current_CGPA` | Current cumulative GPA |
| `Overall_Attendance_Percentage` | Overall attendance % |
| `Internal_Mark_1` / `Internal_Mark_2` | Internal assessment scores |
| `Arrear_Count` | Number of pending arrear subjects |
| `Certification_Count` | Number of certifications earned |
| `Project_Count` | Number of projects completed |
| `Internship_Status` | Paid / Unpaid / Not Started |
| `Placement_Eligible` | Eligible / Ineligible |
| `Sem1_CGPA` … `Sem7_CGPA` | Semester-wise CGPA |
| `Sem1_Attendance_Percentage` … `Sem7_Attendance_Percentage` | Semester-wise attendance |
| `Mentor_Name` | Assigned faculty mentor |
| `Parent_Name`, `Parent_Mobile_No` | Parent contact details |
| `Student_Mobile_No`, `Email_ID` | Student contact details |
